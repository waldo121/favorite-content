# Brainrot tracking system

## Purpose

Keeping track & rewatching of my favorite brainrot content from various platforms

## Stack

- Golang
- sqlite
- htmx
- tailwind

## API Integration

- youtube shorts
    Only "https://www.youtube.com/shorts/<id>" url are supported
- Others TBD


## Running in dev

```shell
go run scripts/database.go 
~/go/bin/air
```
